# Atendimento.Center Frontend

Frontend próprio do **Atendimento.Center**, uma solução Atlas Global.

O projeto utiliza as maquetes aprovadas como base visual para:

- Landing page pública
- Aplicação autenticada
- Caixa de entrada e conversas
- Contactos e agendamentos
- Automação
- IA e base de conhecimento
- Relatórios e configurações

## Stack inicial

- Next.js
- TypeScript
- Tailwind CSS
- Componentes acessíveis
- Integração exclusivamente com o Backend/BFF Atlas

O navegador nunca deve receber credenciais administrativas do Chatwoot, Evolution API ou OpenRouter.
